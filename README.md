# RM_ChatBot
Teams Webhook API를 활용한 RM ChatBot 개발
---

Teams의 Outgoing Webhook API를 이용하여
신속하게 RM 대응 체계 및 통계를 Display하고, 서비스 영향시간을 최소화 시킨다.
